All about a Website





What is a website?



To say in a line a website is simply - “a set of related web pages located under a single domain name”

Think of a website as a book and each individual page is a web page containing some information.

• A website is typically a collection of web pages, images and other elements that are linked together to form a larger and structured document.

• A website can be made up of a single page or it could constitute thousands of pages.

• Each page has its own text, images and other elements.



The Basics of a Website





Making website or web applications (an application working on a website) is referred to as Web Development.

To understand how a website is made, let’s take a look at the very basics of how websites work.

Every single website on the Internet requires just two things to work.
That’s right. Just two.
They are:

* Domain name. This is the unique URL name for your website typically followed by a .com, .org,. online, etc.

* Web host. This is where your website “lives.” It’s the servers you use to store your website’s information.

Combined, these two things allow your website to simply exist on the Internet.



 

What does a website consist of ?





A website has 3 parts:



Front-End:



• If you are a visitor(user) to a website the part of the website which you can access and interact with is called as the frontend.

• A user interacts with the Front-End of the website and perform various activities and the functions the website has.

• The Front-End has a Graphical User Interface / GUI to make it easy for a user to interact.

• Front-end is the client side of the website.

• For example – www.youtube.com has a user-friendly UI design which makes it simpler to use.


The Fornt-End of a website has 3 parts:



For presenting text – HTML or PHP,


• HTML also known as HyperText Markup Language and PHP(Hypertext Preprocessor) are widely used languages for scripting text on a website.


For Stylizing a Website – CSS,


• CSS (Cascading Style Sheets) primarily handles the look and feel of a Website.

• For example – Hover Effects, adding awesome fonts and colors, some visual graphics, etc.


For increasing Interactivity – JavaScript,


• JavaScript provides interactivity to a webpage, handle complex functions and features to increase functionality.

• For example – submission of a form on a button click, Selection of only one option at a time.


Back-End:



• Back-end Development refers to the server-side development.

• It is the term used for the behind-the-scenes activities that happen when performing any action on a website.

• Backend developer focuses on databases, scripting, and the architecture of websites.


Back-End of a website can be build with the help of Web Framework of any Language, like:



• PHP, Python, Ruby, Javascript, Java, Rust, C#, and many more programming languages are used.


Database:



• The database is the storage unit.

• The actual data and content of a website remains stored in a Database.

• The data is stored in a arranged and systemtic manner.

• Example – MySQL, Oracle, DB2Postgres, etc. are some of the DBMS(Database Management System) used for storing data.
